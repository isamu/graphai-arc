## Project Overview

GraphAI-based AI2 ARC challenge solver. Uses GraphAI agents to solve ARC (Abstraction and Reasoning Corpus) tasks.

## Commands

```bash
yarn format     # Format with Prettier
yarn test       # Run ARC solver (npx ts-node)
```

## Architecture

- `ai2_arc/` - ARC challenge solver code and test data
- Uses GraphAI with LLM agents for problem solving
